
# Employee Insights SQL Analysis

## Project Overview
SQL analysis of 300,000+ employee records answering 
5 key business questions around compensation, gender 
equity, and workforce turnover.

## Tools Used
- MySQL
- Employees Database (300K+ records)

## Business Questions Answered
1. Which departments have highest salary growth?
2. Who are the top 3 earners per department?
3. Which employees never received a raise?
4. What is the gender pay gap per department?
5. Which departments have the highest turnover?

## Key SQL Concepts Used
- Window Functions (RANK, LAG, SUM OVER)
- Common Table Expressions (CTEs)
- CASE WHEN pivoting
- Complex multi-table JOINs
- Data filtering and aggregation
